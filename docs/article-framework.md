### 5 分钟跑起来（400 字）—— 实操

```bash
git clone https://github.com/ChanningLua/praxnest
cd praxnest
docker-compose up -d
docker-compose exec praxnest python -m praxnest init --admin-username admin
# 浏览器: http://你的IP:7878
```

### 实战 demo（500 字）—— 一个完整故事

**主角**：一个 5 人小团队（PM / 2 研发 / 1 测试 / 1 运营）做"用户登录"功能。

**故事流**：

1. PM 在 praxnest 里写 PRD（笔记标题：`登录功能 PRD`）
2. PM 选中 PRD 内容 → AI 助手 → **生成测试用例** → 复制到新笔记 `登录功能测试计划`
3. 测试同学跑完一轮，把测试结果 json 粘到 `Sprint 5 测试报告`
4. 选中测试报告 → AI 助手 → **生成执行摘要** → 复制到飞书群发出
5. 第二天测试发现 bug → 在新笔记 `Bug-001 邮箱登录无响应` 写两行现象 → AI 助手 → **套标准模板** → 自动填出复现步骤 / 影响范围 / 期望结果
6. 一个月后做下个功能：研发开始写新 PRD → 团队记忆 sidebar 自动跳出"6 个月前讨论过类似的认证设计"

→ 配 2-3 张 demo 截图
